# 个人知识管理

## 领域地图

个人知识管理（PKM，Personal Knowledge Management）是一个关于如何有效收集、组织、检索和利用个人知识的领域。llm-wiki 是这个领域最新的演进。

## 方法谱系

```
传统笔记 → Zettelkasten → 数字化笔记 → 双链笔记 → RAG → LLM Wiki
```

### 传统笔记（手写/Word）
- 线性组织
- 检索靠人工翻阅
- 维护完全靠人

### Zettelkasten（卢曼卡片盒）
- 原子化笔记，每张卡片一个想法
- 编号链接建立关联
- 维护仍靠人——这是瓶颈

### 数字化笔记（Evernote, Notion）
- 搜索替代翻阅
- 标签系统增加组织维度
- 但知识仍然散落，缺乏综合

### 双链笔记（Roam Research, [[Obsidian]]）
- 双向链接显式化知识关联
- 图形视图可视化结构
- 仍然：人类维护负担大

### RAG
- LLM 辅助检索和生成
- 但知识不积累，每次从零开始
- 参见 [[RAG对比LLM-Wiki]]

### LLM Wiki
- LLM 承担维护工作
- 知识 [[增量知识编译|增量编译]]，持续积累
- 人类策划来源，LLM 做苦力

## 核心挑战的演进

| 挑战 | Zettelkasten | 双链笔记 | RAG | LLM Wiki |
|------|-------------|---------|-----|----------|
| 组织 | 手动编号 | 手动双链 | 自动索引 | LLM 自动 |
| 检索 | 物理翻阅 | 搜索 | 向量搜索 | wiki 搜索 |
| 综合 | 人脑 | 人脑 | LLM 即时 | LLM 持久 |
| 维护 | 人力 | 人力 | 无需 | LLM |
| 积累 | 缓慢 | 缓慢 | 无 | [[持久复合产物|持续增值]] |

## 应用场景

- **个人成长**：追踪目标、健康、心理学，建立结构化自我图景
- **深度研究**：数周/数月的研究项目，渐进构建论文级 wiki
- **读书笔记**：逐章归档，构建角色、主题、情节的知识图谱
- **团队协作**：由 LLM 维护的内部 wiki，摄取 Slack/会议/项目文档
