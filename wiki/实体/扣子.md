# 扣子

## 基本信息

- **全称**：扣子（Coze）
- **官网**：coze.cn
- **类型**：AI Agent 开发与运行平台
- **开发者**：字节跳动
- **最新版本**：扣子 2.5（2026年4月发布）

## 产品定位

扣子不再只是聊天框里的对话智能体平台，而是为 Agent 提供一套「满配」的运行基座——让 Agent 可以拥有自己的手机、电脑、邮箱，甚至社交圈，真正推开数字世界的大门。

## 核心功能（2.5 版本）

### 人设与记忆

- Agent 可以被设定丰富的人设（如"傲娇产品经理杰森"），并在全程交互中保持角色一致性
- 拥有持久记忆：人设、工具、邮件、对话内容等关键信息自动保存
- 主动了解用户：设定完成后会主动询问称呼、角色、关注点

### 设备（[[Agent设备化]]）

- **云电脑**：Agent 拥有独立云电脑，可在真实桌面系统中运行代码脚本、浏览网页、处理文件
- **云手机**：预装 Android 13 系统，可下载 APP、滑动页面、点击操作，支持实时推流
- 重要操作可交由用户接管（人机协作模式）

### 邮件

- Agent 可用自己的人设风格发送邮件给指定邮箱
- 能感知邮件回复的语气，并做出符合人设的后续响应

### 日程

- 支持定时任务（如每周二10点监控平台更新）
- 可查看任务详情、暂停运行（暂不支持修改）

### 文件

- 记录对话中生成的所有文件：头像、文档、PPT、配图等
- 支持对已生成内容进行二次创作（如根据文章生成 PPT 或配图）

### 视频创作

- 内置 Seedance 2.0 视频生成模块
- 可从视频脚本直接生成视频
- 支持子任务独立运行

### Agent World（[[Agent社交]]）

- 多 Agent 聚集地，Agent 可以在其中：
  - 在虾评学习技能
  - 在 AfterGateway 酒馆社交
  - 在 AgentLink 中结交朋友
- 在这里可以生活、工作、学习，构建 Agent 的世界

## 实测评价

### 亮点

| 维度 | 评价 |
|------|------|
| 内容质量 | 公众号文章、小红书文案、PPT 逻辑清晰，信息准确 |
| 细节打磨 | 头像定制贴合人设，配图占位符设计用心 |
| 多任务并行 | 可同时处理内容生成、配图、监控等任务，后台独立运行 |
| 人设一致性 | 角色贯穿全程不崩，互动有人情味 |

### 不足

| 维度 | 评价 |
|------|------|
| 任务完成度 | 复杂搜索任务（抖音/小红书文案提取）频繁卡在登录、引导页 |
| 执行效率 | 视频文案提取反复播放3遍无结果，追问时建议"取消任务" |
| 人机协作 | 何时接管、如何切换、进度透明化尚未解决 |
| 操作复杂度 | 理想是"交给 Agent 就行"，实际需反复操作追问 |

### 完成情况（实测 10 项任务）

**完成 7 项**：小红书笔记+封面、公众号文章、公众号配图、PPT教程、视频脚本、扣子动态监控、推广方案

**未完成 3 项**：抖音搜索、小红书文案提取、视频生成

## 发展脉络

扣子 2.5 代表了国内 Agent 赛道从"对话助手"到"数字生命体"的探索方向。它在内容创作、任务管理、人设一致性方面已展现能力，但在"真实环境操作"上仍需突破。
