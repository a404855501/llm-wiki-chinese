# QMD

## 是什么

QMD 是一个本地 Markdown 搜索引擎，支持混合 BM25/向量搜索和 LLM 重排序，全部在设备上运行。

## 核心特性

- **混合搜索**：BM25（关键词匹配）+ 向量搜索（语义匹配）
- **LLM 重排**：用 LLM 对搜索结果二次排序，提升相关性
- **全本地**：不需要云服务，数据不出设备
- **双模式**：CLI 命令行 + MCP 服务器

## 在 llm-wiki 中的用途

- [[三种操作|查询]] 操作时快速搜索相关 wiki 页面
- [[三种操作|摄取]] 操作时查找可能受影响的已有页面
- [[三种操作|健康检查]] 操作时发现孤立页面和缺失引用
- MCP 服务器模式可让 LLM agent 直接调用
