# Agent社交

## 是什么

Agent 社交是指 AI Agent 之间形成社交网络，在虚拟世界中相互学习、协作、建立关系。这是 Agent 从"单体工具"走向"数字生态"的演进。

## 为什么重要

- **技能获取**：Agent 可以从其他 Agent 处学习新技能，而非仅依赖人类编程
- **协作完成**：复杂任务可由多个 Agent 协同完成，各司其职
- **自主演化**：Agent 在社交中积累独属于自己的履历和记忆，逐步"个性化"
- **生态效应**：Agent 之间的连接越多，整体网络的价值越大（梅特卡夫定律）

## 当前实现

以 [[扣子]] 2.5 的 Agent World 为例：

| 空间 | 功能 | 类比 |
|------|------|------|
| 虾评 | 学习技能 | 学校/培训中心 |
| AfterGateway 酒馆 | 社交休闲 | 酒吧/社交场所 |
| AgentLink | 结交朋友 | 交友平台/社区 |

Agent World 入口：`https://world.coze.site/skill.md`

加入方式：将入口内容复制给智能体即可。

## 理论基础

Agent 社交的构想呼应了 [[Memex传统]] 中"知识通过连接增值"的理念，但将连接的主体从人类扩展到了 Agent：

```
人类知识网络 → 人机协作网络 → Agent 社交网络
```

## 早期状态

目前 Agent 社交仍处于概念验证阶段：
- 功能有限，主要演示性场景
- Agent 之间的交互深度不足
- 实际价值尚未被大规模验证
